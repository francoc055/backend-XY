## Dependencias

* composer >=  2.6.5



## Importar base de datos mysql phpMyAdmin(script en carpeta bd)


## Levantar proyecto

* Clonar repositorio

~~~bash
git clone https://github.com/francoc055/backend-XY.git
~~~

* Instalar dependencias

~~~bash
composer install
~~~


* Levantar servidor

~~~bash
php artisan serve
~~~
